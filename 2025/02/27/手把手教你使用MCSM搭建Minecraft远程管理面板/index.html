手把手教你使用MCSM搭建Minecraft远程管理面板

2025-02-27T12:47:46.000Z2025-02-27T13:11:22.298Z8493 min深圳

一、MCSManager 安装

1. Windows 系统

  1. 从官网下载 Windows 版安装包并解压。
  2. 双击 运行.bat 启动面板，访问 http://服务器IP:23333，使用创建默认账号(管理员)

2. Linux 系统

一键安装脚本（推荐）

安装后访问 http://服务器IP:23333，使用创建默认账号(管理员)

二、创建 Minecraft 服务器

  1. 新建实例
    面板中点击「应用实例」→「新建应用」→ 选择「Minecraft Java 版」

  2. 上传服务端核心或使用实例模板创建
    例如上传 Paper-1.21.jar，配置启动命令：

  3. 修改配置文件

    • 在文件管理器中编辑 eula.txt，将 eula=false 改为 eula=true。
    • 修改 server.properties 中的 server-port=25565（默认端口）。

三、使用 OpenFrp 实现内网穿透

如果你的服务器拥有公网IP，请跳过这一段

1. 注册与配置 OpenFrp

  1. 注册账号
    访问 OpenFrp 官网，完成注册并登录。
  2. 创建隧道
    • 进入「隧道管理」→「创建隧道」。
    • 选择 HTTP 协议，填写以下参数：
      • 本地IP：127.0.0.1（若穿透 MCSManager 面板，本地端口填 23333；若穿透 MC 服务器，填 25565）
      • 本地端口：根据需求选择（见下表）。
      • 远程端口：自动分配或手动指定（需实名认证）。
穿透目标 本地端口 协议类型
MCSManager 面板 23333 HTTP
Minecraft 服务器 25565 TCP
  1. 获取连接信息
    创建成功后，记录「隧道ID」「访问密钥」和「节点地址」（如 frp.openfrp.top:12345）。

2. Windows 系统穿透步骤

  1. 下载客户端
    从 OpenFrp 软件下载页 获取 Windows 版客户端。

  2. 配置隧道

    根据官方文档创建隧道

  3. 启动隧道

    在启动器中找到对应的隧道启动即可~

3. Linux 系统穿透步骤

由于Linux配置过于复杂，请自行参照官方文档进行配置

四、远程连接与测试

  1. 访问 MCSManager 面板
    浏览器输入 http://节点地址:远程端口（例如 http://frp.openfrp.top:54321）。
  2. 连接 Minecraft 服务器
    • 在 Minecraft 多人游戏中输入 节点地址:远程端口（例如 frp.openfrp.top:25565）。
  3. 验证穿透状态
    在 OpenFrp 控制台的「隧道列表」中查看实时流量状态。

五、安全注意事项

  1. 密码安全

    • 不要将面板管理员告知他人
    • OpenFrp 访问密钥定期更新(没必要)

  2. 隧道隔离
    建议为面板和游戏服务器创建独立隧道，避免共用端口

  3. Docker

    使用Docker容器隔开每个实例(如需要用于商业用途)

六、常见问题

问题 解决方案
隧道连接失败 检查本地服务是否运行（如 MCSManager 或 MC 服务器是否启动）
远程访问延迟高 更换 OpenFrp 节点（推荐选择地理距离近的节点）
OpenFrp 客户端报错「auth failed」 确认访问密钥填写正确，或重新生成密钥
Minecraft 无法连接 检查 server.properties 中 online-mode 是否为 false（允许非正版用户）

扩展资源

  • MCSManager 官方文档：https://docs.mcsmanager.com
  • OpenFrp 帮助中心：https://openfrp.wiki
  • Minecraft 服务端配置指南：https://minecraft.fandom.com/wiki/Server.properties

提示：

  • OpenFrp 免费版提供基础穿透服务，若需更高带宽或固定域名，可升级至付费套餐
  • 生产环境建议搭配 Docker 部署，提升安全性和稳定性
