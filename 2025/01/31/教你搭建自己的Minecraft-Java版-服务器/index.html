教你搭建自己的Minecraft(Java版)服务器

2025-01-31T13:34:03.000Z2025-02-07T14:07:35.054Z1.4k5 min深圳

都2025年了,还不会自己开Minecraft服务器?

别担心!这篇教程手把手教你如何开服!

合适的环境

你需要一台可以从公网访问的机器(家用电脑\服务器等,但都要支持从外部直接访问)

家用电脑一般没有公网ip,但不排除本地运营商给你配备了动态公网ip.而购买的云服务器基本配备公网ip

如何查看自己是否拥有公网ip：

首先你需要登录到路由器或者光猫后台,在运行界面找到WAN IP

再去IPW查询自己的IP,如果两个一致,那就是拥有公网IP,可以直接从外部连接到服务器

如果不一样也不要着急,这并不代表你开不了服务器,可以选择购买VPS或者根据下文教程进行端口穿透

性能要足够

一般来说开原版服或者轻量插件服,并且人数不多,最好在CPU2核,内存4G以上.大型插件服和模组服建议4核8G起步(仅仅是起步!特别是模组服真的很吃配置!)

CPU频率,IPC值和单核性能(特别是模组服)越高越好，硬盘读取速度越快越好推荐固态硬盘 耐摔

系统的选择

一般使用Windows或者Linux(新手不推荐,本教程用Windows做示范)

大型服务器使用Linux要比Windows好(因为占用是真的低,Windows动不动吃1-2G)

Windows系统最好是选择WindowsServer,占用要比一般的Windows小

1.7- 推荐 Java8

1.8+ 推荐 Java8/11

1.16+ 推荐 Java11/16

1.18+ 推荐 Java17

如果你的机器已经符合这些要求,就可以进行下一步辣!

选择合适的服务端

现在的服务端多种多样,新手还是推荐从官方原版端或Paper端中选择

不建议新手开MOD服

原版服或者插件服真的要比MOD服流畅很多,而且开服成本较低,

服务器核心大概分为以下几类: 原版端,插件端,模组端

模组端需要用特殊的客户端登入(装有相同MOD),而原版端和插件端任意同版本客户端都可登入(在插件端里甚至是不同版本!有跨版本插件ViaVision可以用!

就连基岩版也可以用Geyser在插件端里面游玩 (更多信息请见：我的世界Geyser服务端介绍及教程（互通服必备 - 哔哩哔哩)

推荐新手使用的核心有这几种:

  • 1.Vanilla端

这是MOJANG官方发布的原版服务端,和原版游戏一模一样,但是不能加装模组和插件

  • 2.Paper端

这是spigot端的下游服务端,优化较好(相比Spigot优化较为激进),但是修复了原版不少BUG特性,可以装插件(说是有极少数插件不兼容,但是我开了两年服到现在还 没见过不兼容的插件),不能装模组

  • 3.Vanilla+Forge/Fabic,经典模组端

新手上手难度较大(但是是最适合新手的模组端),只能装模组,不能装插件,需要特定客户端登入

  • 4.CatServer端

CatServer是一款高性能且高度兼容的服务器核心，它集成了Forge、Bukkit和Spigot的功能，支持1.12.2、1.16.5以及1.18.2等游戏版本。这个项目由Luohuayu发起，采用GPL-3.0许可证进行分发，主要编程语言为Java，适合想要自定义服务器环境、添加MOD和插件的玩家和技术爱好者

不建议新手使用！！！

但是本身是一个很厉害的端!

推荐一个很好的端: Purpur端

他是Paper端的优化版，主要差别在性能和占用两方面上

对性能有较大的提升

开服!!!

这一步非常简单,只要把你选好的核心下载下来,然后放到一个空的文件夹里面,创建好启动脚本即可.

首先新建一个空的文件夹,名字随便起

然后把下载的服务端核心丢进去

右键新建一个文本文档,双击打开,里面像这样编辑

@ECHO OFF

java -Xmx1G -Xms2G -jar 服务端核心名称.jar

pause

记得在查看文件的时候一定要勾选上文件扩展名,方便后续操作.(如果有不会的可以自行搜索,这个每个版本的Windows开启位置都不一样)

最后保存(Ctrl+S)然后把.txt文件修改为.bat文件就好啦

双击.bat文件,你的服务器就开好了!

这时候你会发现,服务器打开了但是没有完全打开

这就需要你回到服务器的根目录(也就是你创建的那个放服务器核心的文件夹),发现新生成了一个叫eula.txt的文件

然后将其中的 eula=false 改为 eula=true,这代表你同意了Mojang的协议

保存并关闭,再次双击打开.bat文件,你的服务器就可以正常开启啦!

想停止的话在控制台输入stop就可以啦!

添加插件

本节教程只适用于插件端 - Paper，Catserver，Spigot，Bukkit等服务端

“大多数”插件服务器的根目录下都有一个plugins文件夹（如果没有，请手动创建）

将想要安装插件的jar文件扔进去就行了

然后就没了。。

Paper/Spigot/Bukkit用户看过来 –> PaperMC的插件库

ESS插件 - Essentials - Paper Plugin | Hangar

MV-Core插件 - Multiverse-Core - Paper Plugin | Hangar
