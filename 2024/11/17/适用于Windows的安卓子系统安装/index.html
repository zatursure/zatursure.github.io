适用于Windows的安卓子系统安装

2024-11-17T03:26:38.000Z2024-11-17T08:55:08.847Z3821 min深圳

WSA（适用于Windows的Android™️子系统），全称Windows Subsystem for Android，它能够让你在Windows 11系统中安装和使用Android应用程序。

官方开发人员指南链接：适用于 Android™️ 的 Windows 子系统 | Microsoft Docs（https://docs.microsoft.com/zh-cn/windows/android/wsa/）

一. 安装环境

设备要求

确保设备已安装Windows 11（Win11的安装请自行搜索）

内存：8 GB (最低) 16 GB (建议)

处理器：Intel Core i3 第 8 代 (最低)/AMD Ryzen 3000 (最低) /高通 Snapdragon 8c (最低)

在Windows 11电脑上启用虚拟化

打开 开始，输入 Windows 功能，然后从结果列表中打开 启用或关闭 Windows 功能

在刚刚打开的 Windows 功能窗口中，找到 Hyper-V、Windows虚拟机监控程序平台和 虚拟机平台并勾选

点击 确定。 可能需要重启电脑

二.安装并配置WSA

下载安装包

由于中国用户无法在微软应用商店安装，所以需要下载安装包进行安装

访问Microsoft Store 的在线链接生成器，获取方式选择ProductId，通道选择Fast，搜索框中填入9P3395VX91NR

在下面加载出的文件列表中, 找到名称为如下格式的文件, 下载版本最新的那个, 大约 1.37GB

MicrosoftCorporationII.WindowsSubsystemForAndroid_x.x.x.x_neutral_~_8wekyb3d8bbwe.msixbundle

等待浏览器下载完成后安装

使用WSAToolBox管理WSA

使用微软应用商店下载WSAToolBox —-> 传送门

工具箱中的内容自己摸索一下，然后。。。这篇文章就结束啦
